## Live Sever 사용법

![방법1](https://github.com/SiRyung/coursegraph-js/blob/main/img/liveserver1.PNG)<br><br>
vscode 좌측 확장에서 Live Server를 검색합니다.

---
![방법2](https://github.com/SiRyung/coursegraph-js/blob/main/img/liveserver2.PNG)<br><br>
사진과 같이 Live Sever과 표시되면 설치버튼을 눌러줍니다.

---
![방법3](https://github.com/SiRyung/coursegraph-js/blob/main/img/liveserver3.PNG)<br><br>
생략해도 됩니다. (오류 발생시 방법 권장)
Live Server 관리 버튼을 눌러줍니다.

---
![방법4](https://github.com/SiRyung/coursegraph-js/blob/main/img/liveserver4.PNG)<br><br>
하단으로 스크롤 하여 Live Sever > Settings: Use Local IP에 해당하는 부분인
Use local IP as host를 활성화 시켜줍니다.

---
![방법5](https://github.com/SiRyung/coursegraph-js/blob/main/img/liveserver5.PNG)<br><br>
그림과 같이 우클릭으로 Open with Live Server를 눌러 실시간으로 반영되는 라이브 서버를 활성화 시킬 수 있습니다.
종료를 원한다면 Stop Live Server를 클릭하면 라이브 서버가 비활성화 됩니다.
